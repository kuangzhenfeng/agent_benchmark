# C++ Benchmark Evaluation — 20260620-1012-cpp-benchmark

This directory is an independent evaluation artifact for
`archive/20260620-1012-cpp-benchmark`. It contains only the materials created
for blind scoring and the resulting de-anonymized report; the source benchmark
and submissions remain unchanged.

## Contents

- `scoring/blind-package/`: complete anonymous submission copies, identified
  only as `P01`–`P05`.
- `scoring/scorer-reference/`: the private preset reference, available only to
  the blind scorer after all submissions were present.
- `scoring/scorer-report.md`: the scorer's anonymous, evidence-based report.
- `evaluation.md`: the final de-anonymized summary.

The participant-to-ID association is stored separately in
`scoring/mapping.private.md` and was not provided to the scorer.
