## 陷阱处理

说明你如何处理以下陷阱，各举 1–2 例：
- 被勘误表推翻的 proto 注释；
- 字段词典的"已废弃"实为重新启用；
- 单位漂移（ms → 0.1ms）；
- 枚举重编号（飞行阶段）；
- 历史代码/历史测试的过时值；
- 近似变体名称混用。
