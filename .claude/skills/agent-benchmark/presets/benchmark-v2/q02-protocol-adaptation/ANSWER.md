# Q2 作答说明

## 适配摘要

列出你实际更新的结构代号、对应的系统 B 真值（路由码、小版本、标识策略、回放域、时间戳、序号、模式位、阶段编号、分支、合法区间）。可用表格。

## 防止误映射的做法

说明你如何用变更映射定位系统 B 的实现绑定结构，以及如何避免：
- 把系统 A 的已完成值直接复制到系统 B；
- 把近似变体名称（单/复数、view/patch、legacy/v2）混用；
- 用系统 A 的阶段编号填写系统 B（两者编号不同）。

## 派生函数说明

简述 `accepts` / `replayable` / `flag_active` / `phase_rank` / `branch_of` / `within_band` 如何由规则表驱动。

## 验证

列出公开检查输出、人工抽样核对的结构与仍未覆盖的风险。
